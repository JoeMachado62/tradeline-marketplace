Prerequisites

Please follow below steps to install and setup all prerequisites:

  • Node.js

    To use our build tools and take full advantage of Finazze's customization options, you will need to download and install Node.js. If you do not have Node.js installed already, you can download the package installer from the official website. We recommend that you download the stable version of Node.js (LTS) to ensure the best possible experience. Download Node.js

Start

Command Description
npm i or yarn This would install all required dependencies node_modules folder..
npm run dev or yarn dev Runs the project locally, starts the vite and node development server.
